前言 :

DOC.txt是原本CDemo.c編譯成CDemo.exe前的使用解說。

裡面的名稱結構是使用C語言的方式，雖然slvs.py已經幫忙轉換了裡面的名稱，但是仍有些許函式名稱變化。

所以紀錄一下在Python中如何使用這些功能，並稍微翻譯其內容；而手冊中不清楚的地方也會補充。

預計讓這篇文章達到易使用和易理解的功能。

預期運行結果 :

若要使用SLVS資料庫，必須透過slvs.py介面來轉換。

在你的Python程式中使用以下指令：

#導入slvs.py
from slvs import *

#導入Solid Python
#沒有使用就不需要導入
import solid

#創造你的系統，以及支援單位數量
#支援單位數量上限端看Python的記憶體數量
#預設單位數為50
sys = System(20)

...

欲執行解題，Python必須使用下列命令：

#執行解題
sys.solve()
#直接命令
Slvs_Solve(sys, g)

#結果回報的值如下
if (sys.result == SLVS_RESULT_OKAY):
    ...
elif (sys.result == SLVS_RESULT_INCONSISTENT):
    ...
elif (sys.result == SLVS_RESULT_DIDNT_CONVERGE):
    ...
elif (sys.result == SLVS_RESULT_TOO_MANY_UNKNOWNS):
    ...

在執行解題程式後，會出現三種結果：

  1. 所有約束都能滿足，會回傳SLVS_RESULT_OKAY，param中的內容也會更新。

  2. 解題程式能夠證明約束條件互相牴觸，會傳回SLVS_RESULT_INCONSISTENT，相關約束的列表會產生在failed列表中。

  3. 解題程式能夠證明約束條件沒有互相牴觸，但也沒有完全約束，會傳回SLVS_RESULT_DIDNT_CONVERGE，相關約束的列表會產生在failed列表中。

最後能調用以下值來檢查結果。

#實體結果
#格式為%.3f
sys.get_param(0).val

#約束結果
#格式為%lu
sys.failed[i]

而過於嚴重的錯誤會回傳SLVS_RESULT_TOO_MANY_UNKNOWNS，必須檢查整個程式。

TYPES OF ENTITIES

SLVS_E

實體的類型，屬於物件的部分。

  1. 3D 此物件屬於3D類型。

  2. 2D 此物件屬於2D類型。

  3. Point 此物件屬於點。

  4. Vector 此物件屬於向量。

  5. Line 此物件屬於線段。

  6. Curve 此物件屬於曲線。

  7. Arc 此物件屬於圓弧。

實體可以馬上建立物件並用傳統結構的方式導入。

建立物件後也能用轉換好的slvs.py函式呼叫。

或是使用直接指令建立物件。

輸入值後方加入, sys，以傳入系統清單sys中；亦可用sys.add_加入以下支援的實體，而取出的編號取決於註冊的順序。

Point in 3D 3D Point

SLVS_E_POINT_IN_3D

使用指令：

#標準
a = sys.add_param(10.0) #X軸座標
b = sys.add_param(20.0) #Y軸座標
c = sys.add_param(30.0) #Z軸座標
p = sys.add_point3d(a, b, c)
#直接命令
Slvs_MakePoint3d(h, group, x, y, z)
#簡化
p = Point3d(a, b, c)

p代表空間中的輸入點。

Point in 2D 2D Point

SLVS_E_POINT_IN_2D

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
a = sys.add_param(10.0) #U軸座標
b = sys.add_param(20.0) #V軸座標
p = sys.add_point2d(wrkpl, a, b)
#直接命令
Slvs_MakePoint2d(h, g, wrkpl, u, v)
#簡化
p = Point2d(wplane, a, b)

p代表工作平面上的輸入點。

Normal in 3D 3D Vector

SLVS_E_NORMAL_IN_3D

使用指令：

#標準
w = sys.add_param(10.0) #實數座標
a = sys.add_param(20.0) #X軸座標
b = sys.add_param(30.0) #Y軸座標
c = sys.add_param(40.0) #Z軸座標
p = Normal3d(w, a, b, c)
#直接命令
Slvs_MakeNormal3d(h, g, qw, qx, qy, qz)

p代表空間中的一條法線。

在SolveSpace中，法線是由一個3x3的旋轉矩陣從基本的坐標系到一個新的架構，是由單位四元數定義的。

可以想像這個四元數代表一個通過原點的平面。

這個平面跟三個向量相關：基礎向量U和V在平面上、而法線N則垂直它，相當於[ U V N ]'。

因此U、N、N都有單位長度，而他們都正交，所以：

1
2
3
U cross V = N
V cross N = U
N cross U = V

轉換函式Slvs_Quaternion提供了在U、V、N和單位四元數之間轉換。

單位四元數只有三個自由度，但是要在四個參數中指定。因此一個額外的約束產生了，就是：

w^2 + x^2 + y^2 + z^2 = 1

Make a Quaternion for a Work Plane Tips

要定義一個由U[1 0 0]向量（等同X軸）和V[0 1 0]向量（等同Y軸）組成的平面，是需要一條3D法線的。

若是不會計算四元數，可以利用下列函式達成：

#函式轉換四元數
qw, qx, qy, qz = MakeQuaternion(1, 0, 0, 0, 1, 0)

#帶入3D法線的函式中
normal = Normal3d(Param(qw), Param(qx), Param(qy), Param(qz), sys)

#做成工作平面
o = Point3d(Param(0.0), Param(0.0), Param(0.0))
plane = Workplane(o, normal)

這個公式在製作2D平面時非常方便。

以下是不同平面參考：

1
2
3
4
5
6
7
8
XY平面：[1 0 0], [0 1 0]
MakeQuaternion(1, 0, 0, 0, 1, 0)

YZ平面：[0 1 0], [0 0 1]
MakeQuaternion(0, 1, 0, 0, 0, 1)

XZ平面：[1 0 0], [0 0 1]
MakeQuaternion(1, 0, 0, 0, 0, 1)

Distance 2D

SLVS_E_DISTANCE

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
r = sys.add_param(10.0) #距離d
p = Distance(wplane, r)

p代表與工作平面平行的距離。

Work Plane 3D

SLVS_E_WORKPLANE

使用指令：

#標準
normal = Normal3d(1, 0, 0, 0) #某個法線
o = Point3d(10, 10, 10) #某個點
p = Workplane(o, normal)
#直接命令
Slvs_MakeWorkplane(h, g, o, normal)

p代表通過原點，與法線垂直的工作平面。

Line Segment Line

SLVS_E_LINE_SEGMENT

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
a = Point3d(0, 0, 0) #某個3D點
b = Point3d(10, 10, 10) #某個3D點
c = Point2d(wplane, 0, 0) #某個2D點
d = Point2d(wplane, 10, 10) #某個2D點
p = LineSegment2d(wrkpl, c, d)
p = LineSegment3d(a, b)
#直接命令
Slvs_MakeLineSegment(h, group, wrkpl, ptA, ptB)

這個函式在直接命令上是2D的方式繪出線條。

p代表兩個點之間的連線。

Cubic 3D Curve

SLVS_E_CUBIC

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
a = Point3d(0, 0, 0) #第零個點
b = Point3d(10, 0, 0) #第一個點
c = Point3d(0, 10, 0) #第二個點
d = Point3d(0, 0, 10) #第三個點
p = Cubic(wrkpl, a, b, c, d)
#直接命令
Slvs_MakeCubic(h, group, wrkpl, pt0, pt1, pt2, pt3)

p代表四個點構成，可能帶有貝茲曲線的立方體。

四個點可以是四個3D點，也可以是四個2D和工作平面構成。

四個點的公式如下：

#公式（t從0積分到1）
p(t) = P0*(1 - t)^3 + 3*P1*(1 - t)^2*t + 3*P2*(1 - t)*t^2 + P3*t^3

Circle 2D Arc

SLVS_E_CIRCLE

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
normal = Normal3d(1, 0, 0, 0) #某條3D法線
o = Point2d(wplane, 0, 0) #圓心
r = Distance(wplane, 20) #半徑
p = Circle(wrkpl, normal, o, r)
#直接命令
Slvs_MakeCircle(h, group, wrkpl, center, normal, radius)

p代表與一條法線垂直（可選：或與一個平面平行），由圓心和半徑構成的完整圓。

Arc of Circle 2D Arc

SLVS_E_ARC_OF_CIRCLE

使用指令：

#標準
wrkpl = Workplane(p1, wnormal) #某個工作平面
o = Point2d(wplane, 0, 0) #圓心
s = Point2d(wplane, 50, 0) #起點
e = Point2d(wplane, 0, 50) #終點
normal = Normal3d(1, 0, 0, 0) #某條3D法線
p = ArcOfCircle(wrkpl, normal, o, s, e)
#直接命令
Slvs_MakeArcOfCircle(h, group, wrkpl, normal, center, start, end)

p代表三個點與一條法線垂直（可選：或與一個平面平行）的不完全圓。

TYPES OF CONSTRAINTS

SLVS_C

約束的類型，大部分的指令都會自動判斷狀況。

  1. 3D 此種約束只能用在3D或投影到平面上。

  2. 2D 此種約束只能用在平面上。

  3. 未標示 3D 或 2D 則兩者皆可。

  4. Val 此種約束必須設定額外的參考值。

輸入值最前端加入平面名稱可以規定為投影到平面的運算。

約束的直接命令：

Slvs_MakeConstraint((h, group, type, wrkpl, valA, ptA, ptB, entityA, entityB)

輸入值後方加入, sys，以傳入系統清單sys中；或是使用以下sys.add_命令加入，而取出的編號取決於註冊的順序。

1
add_constraint( )

＊若沒有使用sys的註冊，將會被警告註冊值不可為Null。

Point to Point Distance 3D Val

SLVS_C_PT_PT_DISTANCE

使用指令：

a = Point3d(0, 0, 0) #第一個點
b = Point3d(10, 10, 10) #第二個點
Constraint.distance(20, a, b)

約束兩個空間點的距離，非硬性，必要時會自動調整距離。

Projected Point Distance Val

SLVS_C_PROJ_PT_DISTANCE

使用指令：

a = Point3d(0, 0, 0) #第一個點
b = Point3d(10, 10, 10) #第二個點
normal = Normal3d(1, 0, 0, 0) #某條3D法線
Constraint.distance(50, a, b, normal)

第一點到線段上的投影，與第二點的距離，是硬性的約束。

Points Coincident 3D

SLVS_C_POINTS_COINCIDENT

使用指令：

a = Point3d(0, 0, 0) #第一個點
b = Point3d(10, 10, 10) #第二個點
Constraint.coincident(a, b)

兩個點的座標相等。（即：在彼此之上）

Point to Plane Distance Val

SLVS_C_PT_PLANE_DISTANCE

使用指令：

a = Point3d(0, 0, 0) #第一個點
wrkpl = Workplane(p1, wnormal) #某個工作平面
Constraint.distance(50, a, wrkpl)

從一個點到一個平面的距離，是硬性的約束。

距離值的正負會規定點在平面之上或之下。

Point to Line Distance 3D Val

SLVS_C_PT_LINE_DISTANCE

使用指令：

a = Point3d(0, 0, 0) #第一個點
normal = Normal3d(1, 0, 0, 0) #某條3D法線
Constraint.distance(50, a, normal)

從一個點到一條直線的距離，假如約束是投影的，會變成硬性約束。

距離值的正負會規定點在線段之上或之下。

若約束是套用在3D空間，距離則會變成非硬性約束。

Point in Plane

SLVS_C_PT_IN_PLANE

使用指令：

wrkpl = Workplane(p1, wnormal) #某個工作平面
a = Point3d(0, 0, 0) #第一個點
Constraint.on(wrkpl, a)

一個點會被固定在一個平面上。

Point on Plane 3D

SLVS_C_PT_ON_LINE

使用指令：

a = Point3d(0, 0, 0) #第一個點
normal = Normal3d(1, 0, 0, 0) #某條3D法線
Constraint.on(normal, a)

一個點會被固定在一條法線或直線上。

注意此約束會在投影到一個平面上時移除一個自由度；而在3D空間時變成兩個。

Equal Angle 3D

SLVS_C_EQUAL_ANGLE

使用指令：

#功能建構中

第一條線段和第二條線段的夾角等於第三條線段和第四條線段的夾角。

Equal Length of Line to Arc 3D

SLVS_C_EQUAL_LINE_ARC_LEN

使用指令：

a = LineSegment3d(x, y) #某條3D線段
b = ArcOfCircle(wrkpl, normal, o, s, e) #某段圓弧
Constraint.coincident(a, b)

一條線段長度和另一段圓弧長度相等。

Symmetric 3D

SLVS_C_SYMMETRIC

使用指令：

#功能建構中

第一個點會和第二點對稱於一個工作平面。

意味著兩個點到平面的距離都相等，而兩點的連線可以做為平面的法線。

Horizontal and Vertical 2D

SLVS_C_SYMMETRIC_HORIZ

SLVS_C_SYMMETRIC_VERT

使用指令：

a = Point2d(wplane, 0, 0) #第一個點
b = Point2d(wplane, 20, 20) #第二個點
c = LineSegment2d(wplane, a, b) #兩個點所連成的線段
Constraint.horizontal(wplane, a, b)
Constraint.vertical(wplane, a, b)
Constraint.horizontal(wplane, c)
Constraint.vertical(wplane, c)

兩個輸入的點會互相對稱水平軸或垂直軸。

亦或規定一條線段的水平或垂直。

Diameter Val

SLVS_C_DIAMETER

使用指令：

pi = math.pi
a = ArcOfCircle(wrkpl, normal, o, s, e) #某段圓弧
Constraint.diameter(40*pi, a)

一段圓或圓弧的周長或弧長。

Point on Circle

SLVS_C_PT_ON_CIRCLE

使用指令：

a = Point3d(0, 0, 0) #某個點
b = ArcOfCircle(wrkpl, normal, o, s, e) #某段圓弧
Constraint.on(a, b)

一個點會被固定在一個圓或一段圓弧上。

Same Orientation

SLVS_C_SAME_ORIENTATION

使用指令：

#功能建構中

兩條法線的方向相等。

Angle 3D Val

SLVS_C_ANGLE

使用指令：

#功能建構中

兩條線段的夾角，定義值單位為度。

公式如下（A和B為兩條線的向量）：

(A dot B)/(|A||B|) = cos(valA)

注意這裡的值定義比較不明確，無論正負90度都會是相同的結果。

Perpendicular 3D

SLVS_C_PERPENDICULAR

使用指令：

#功能建構中

源自約束SLVS_C_ANGLE。

這個約束是定義角度約束為90度的情況。

Parallel 3D

SLVS_C_PARALLEL

使用指令：

#功能建構中

源自約束SLVS_C_ANGLE。

這個約束是定義角度約束為平行的情況。

注意此約束會在投影到一個平面上時移除一個自由度；而在3D空間時變成兩個。

Arc and Line Tangent 2D Val

SLVS_C_ARC_LINE_TANGENT

使用指令：

#功能建構中

一段圓或圓弧和一條線段相切。

若真假值是False，則圓弧的切點會位於它的起點s；反之若為True，則會位於它的終點e。

Cubic and Line Tangent 3D Val

SLVS_C_CUBIC_LINE_TANGENT

使用指令：

#功能建構中

一個可能帶有貝茲曲線的立方體和一條線段相切。

若真假值是False，則貝茲曲線的切點會位於它的起點w；反之若為True，則會位於它的終點k。

Curve and Curve Tangent 2D Val

SLVS_C_CURVE_CURVE_TANGENT

使用指令：

#功能建構中

源自約束SLVS_C_ARC_LINE_TANGENT。

兩個曲線相切，雙方可以任意為圓弧或是立方體。

若真假值是False，則圓弧的切點會位於它的起點s，貝茲曲線的切點會位於它的起點w；反之若為True，則會位於它的終點e或k。

Equal Radius

SLVS_C_EQUAL_RADIUS

使用指令：

a = ArcOfCircle(wrkpl, normal, o, s, e) #第一段圓弧
b = ArcOfCircle(wrkpl, normal, p, q, r) #第二段圓弧
Constraint.equal_radius(a, b)

兩個圓或圓弧的半徑相等。

Where Dragged 3D

SLVS_C_WHERE_DRAGGED

使用指令：

a = a = Point3d(0, 0, 0) #某個點
sys.set_dragged(a)

命令約束盡量固定這個點，取而代之的是移動周遭的項目。

這個點被固定後移動量會較小。
