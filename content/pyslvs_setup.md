安裝
===

以下將介紹如何在不同作業系統中運行 Pyslvs 套件。

Windows 平台
---

Windows 平台的版本為可獨立執行檔案，下載後雙擊即可啟動 Pyslvs。

Ubuntu 平台
---

Ubuntu 平台上採用 Debian package 安裝。

透過 dpkg 套件將安裝檔的內容複製到系統中。

```bash
sudo dpkg -i pyslvs_vX.X_ubuntu_x64.deb
```

接著透過指令啟動 Pyslvs。

```bash
pyslvs
```
